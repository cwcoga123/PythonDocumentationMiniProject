Overview

What is Python?

Python is a high-level, interpreted programming language known for its readability and ease of use. Created by Guido van Rossum and first released in 1991, Python emphasizes code readability with its notable use of significant indentation.

Here are five important bullet points about Python:

  • Readable and Maintainable Code.
  • Extensive Standard Library.
  • Support for Multiple Programming Paradigms.
  • Dynamic Typing and Memory Management.
  • Vast Ecosystem and Community.
Syntax

Variables and Data Types

Python uses dynamic typing, meaning you don't need to declare the type of a variable explicitly.

x = 10 # Integer
y = 3.14 # Float
name = "Alice" # String
is_active = True # Boolean

Basic Operations

a = 5 + 3 # Addition
b = 5 - 3 # Subtraction
c = 5 * 3 # Multiplication
d = 5 / 3 # Division
e = 5 // 3 # Floor division
f = 5 % 3 # Modulus
g = 5 ** 3 # Exponentiation
Data Types

Numbers

  • Integers: Whole numbers, e.g., '10', '-3'
  • Floats: Decimal numbers, e.g., '3.14', '-2.7'

Strings

Strings are sequences of characters enclosed in quotes.

s = "Hello, World!"

Lists

Lists are ordered collections of items, which can be of different types.

fruits = ["apple", "banana", "cherry"]

Tuples

Tuples are similar to lists but are immutable.

coordinates = (10, 20)

Dictionaries

Dictionaries are collections of key-value pairs.

student = { "name": "Alice", "age": 25, "courses": ["Math", "Science"] }

Sets

Sets are unordered collections of unique items.

unique_numbers = {1, 2, 3, 4, 5}
Control Flow

Conditional Statements

if x > 0: print("Positive") elif x == 0: print("Zero") else: print("Negative")

For Loop

for i in range(5): print(i)

While Loop

count = 0 while count < 5: print(count) count += 1
Functions

Functions are blocks of reusable code that perform a specific task.

Defining a Function

def greet(name): return f"Hello, {name}!"

Calling a Function

message = greet("Alice") print(message) # Output: Hello, Alice!
